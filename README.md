# Homebrew Tap for SBOMHub CLI

This repository contains the Homebrew formula for installing [SBOMHub CLI](https://github.com/youichi-uda/sbomhub-cli).

## Installation

```bash
brew tap sbomhub/tap
brew install sbomhub
```

## Update

```bash
brew upgrade sbomhub
```

## Formula

The formula is automatically updated when a new release is published on the sbomhub-cli repository.
